To run this program the following must be installed via these pip commands within python terminal:

pip3 install Pillow
pip3 install pytesseract
pip3 install opencv-python

If you run into the error:
"cv2 module not found, DLL missing.. or something of that sort."
Try moving the included file: 'opencv_python-3.3.1-cp36-cp36m-win32.whl',
into the project directory. Then call: pip install opencv_python-3.3.1-cp36-cp36m-win32.whl
